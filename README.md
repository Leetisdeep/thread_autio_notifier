# Disrcord Thread_Create Audio Notifier

Plays an audio from sounds folder when thread is crated at targeted channel

## Quick Setup

After installing/cloning the repository:

1. **Open terminal/bash** in the project directory
2. **Copy environment file**: `cp .env.example .env`
3. **Edit `.env` file** with your configuration values
4. **Install dependencies**: `npm install`
5. **Start the application**: `npm start`

## Environment Configuration

Make sure to configure your `.env` file with appropriate values

## Need Help?

- Run `npm install` if you encounter module errors
